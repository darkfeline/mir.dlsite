API for DLsite.

Before running any other make command, run::

  $ pipenv install --dev

To build an installable wheel, run::

  $ pipenv run make wheel

To build a source distribution, run::

  $ pipenv run make sdist

To run tests, run::

  $ pipenv run make check

To build docs, run::

  $ pipenv run make html
